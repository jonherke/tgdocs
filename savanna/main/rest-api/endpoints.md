Type | Name | Description | Schema  
---|---|---|---

---|---|---

Name | Schema  
---|---

**ErrorDetails** _optional_ |  < [middleware.Error](https://docs.tigergraph.com/savanna/main/rest-api/definitions#_middleware_error) > array
