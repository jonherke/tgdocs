# Installing pyTigerGraph
### Contents
  * [Prerequisites](https://docs.tigergraph.com/pytigergraph/1.6/getting-started/install#_prerequisites)

  * [Install pyTigerGraph[gds]](https://docs.tigergraph.com/pytigergraph/1.6/getting-started/install#_install_pytigergraphgds)
  * [Import pyTigerGraph](https://docs.tigergraph.com/pytigergraph/1.6/getting-started/install#_import_pytigergraph)


This page walks you through installing pyTigerGraph on your machine.
## [](https://docs.tigergraph.com/pytigergraph/1.6/getting-started/install#_prerequisites)Prerequisites
  * Python 3.0 or higher.
  * If you are installing `pyTigerGraph[gds]`, make sure [`torch` is installed](https://pytorch.org/get-started/locally/) in your Python environment.


## [](https://docs.tigergraph.com/pytigergraph/1.6/getting-started/install#_install_pytigergraph)Install _pyTigerGraph_
To download _pyTigerGraph_ , run the following command in the command line or use the appropriate tool of your development environment (anaconda, PyCharm, etc.).:
```
pip3 install pyTigerGraph
![copy icon](https://docs.tigergraph.com/_/img/octicons-16.svg#view-clippy)Copied!

```

## [](https://docs.tigergraph.com/pytigergraph/1.6/getting-started/install#_install_pytigergraphgds)Install _pyTigerGraph[gds]_
To utilize [pyTigerGraph’s Graph Data Science (GDS) functionality](https://docs.tigergraph.com/pytigergraph/1.6/gds/), run the following command to install `pyTigerGraph[gds]`.
```
pip3 install 'pyTigerGraph[gds]'
![copy icon](https://docs.tigergraph.com/_/img/octicons-16.svg#view-clippy)Copied!

```

Certain GDS classes, such as [data loaders](https://docs.tigergraph.com/pytigergraph/1.6/gds/dataloaders), can produce output in PyTorch Geometric (PyG) or Deep Graph Library (DGL) formats. To support those formats, you need to have installed the respective packages in your Python environment:
  * [Install Deep Graph Library](https://www.dgl.ai/pages/start.html).
  * [Install pyTorch Geometric](https://pytorch-geometric.readthedocs.io/en/latest/notes/installation.html) .


It is optional to install either of the aforementioned packages. If you don’t install either of the packages, you can still use the GDS functions, and output results in panda frames.
## [](https://docs.tigergraph.com/pytigergraph/1.6/getting-started/install#_import_pytigergraph)Import `pyTigerGraph`
Once the package is installed, you can import it like any other Python package:
```
import pyTigerGraph as tg
python![copy icon](https://docs.tigergraph.com/_/img/octicons-16.svg#view-clippy)Copied!

```
